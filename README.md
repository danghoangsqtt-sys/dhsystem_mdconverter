# Local Markdown Converter (Docling)

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Status](https://img.shields.io/badge/status-completed-success.svg)

Ứng dụng chuyển đổi tài liệu `PDF`, `DOCX`, `DOC`, `PPTX` sang định dạng `Markdown` tối ưu cho AI sử dụng công cụ `docling`. Được phát triển với kiến trúc `DHSYSTEM-TEMPLATE`.

## Cấu trúc Dự án
- `backend/`: Server FastAPI (Python) gọi thư viện `docling` xử lý file.
- `frontend/`: Giao diện React Vite. (Đã được build tĩnh để phục vụ 1-click start).
- `data/`: Thư mục chứa file tải lên (`uploads/`) và kết quả chuyển đổi (`outputs/`). Toàn bộ dữ liệu nằm cục bộ tại đây.
- `docling-env/`: Thư mục môi trường ảo Python chứa thư viện `docling`.
- `start.bat`: Script khởi động 1-click.

## Hướng dẫn Khởi chạy (1-Click)
1. Đảm bảo bạn đã có sẵn Python trong thư mục `docling-env`.
2. Bấm đúp vào tệp `start.bat`.
3. Script sẽ tự động:
   - Cài đặt thư viện phụ thuộc cho Backend (nếu chưa có).
   - Mở cửa sổ Console chạy Server (FastAPI).
   - Tự động mở trình duyệt ở địa chỉ http://localhost:8000
4. Mọi tài liệu chuyển đổi sẽ được bảo mật và lưu trữ trong `data/outputs/`.

## Phát triển (Dành cho Developer)
- Chạy backend: `cd backend` -> `python run_server.py`
- Chạy frontend: `cd frontend` -> `npm run dev`
