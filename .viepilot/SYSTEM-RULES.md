# Quy Tắc Hệ Thống (System Rules)

1. Tuân thủ kiến trúc tách biệt hoàn toàn Frontend/Backend.
2. Code Frontend dùng TypeScript, code Backend dùng Python.
3. Không sửa các files template của dự án mẹ `DHSYSTEM-TEMPLATE` mà phải áp dụng nó vào dự án hiện tại.
4. Giao diện ưu tiên đơn giản, tuân thủ `05-DESIGN_STANDARD.md`.
5. Tích hợp Agentation trong môi trường dev (`04-AGENTATION_INTEGRATION.md`).
