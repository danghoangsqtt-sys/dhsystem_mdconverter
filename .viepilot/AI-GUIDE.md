# Hướng Dẫn AI (AI Guide)

## Bối Cảnh Nhanh
- Ứng dụng local, chạy FastAPI backend và React frontend.
- Cấu trúc thư mục chia `backend/` và `frontend/`.
- Không lưu file ở ổ C:/, chỉ lưu ở mục `data/` ngang hàng dự án.

## File Cốt Lõi
- `01-PROJECT_ARCHITECTURE_TEMPLATE.md` (chuẩn kiến trúc).
- `04-AGENTATION_INTEGRATION.md` (Agentation integration).
- `05-DESIGN_STANDARD.md` (Quy chuẩn giao diện).
